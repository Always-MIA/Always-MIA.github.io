# Always-MIA.github.io - Personal Portfolio Website – Mia Singleton
My beginner styled website

## Overview
This project is a personal portfolio website created using HTML and CSS. It was developed as part of an introductory web development assignment to demonstrate foundational skills in structuring webpages, applying external styling, and publishing a live site using GitHub Pages.


The website presents basic information about my academic background, career interests in computing applications, and personal hobbies. It serves as an early-stage digital portfolio that will continue to evolve as my technical skills grow.


## Purpose
The purpose of this project is to practice core web development concepts, including semantic HTML structure, external CSS styling, and version control using Git. It also provides experience deploying a live website through GitHub Pages.

## Project Goals
- Build a structured personal webpage using semantic HTML
- Apply consistent styling using an external CSS file
- Learn version control through multiple commits
- Publish a live website using GitHub Pages
